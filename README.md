# Parking Guidance System

## Overview

A Java-based Parking Guidance and Management System designed to manage parking operations efficiently through user roles, ticket management, parking spot allocation, payment tracking, and reporting.

The project follows a layered architecture and applies Object-Oriented Programming principles to separate business logic, data storage, and user interfaces.

## Features

- User Management (Admin & Customer)
- Parking Spot Allocation
- Ticket Generation and Tracking
- Entry Station Operations
- Exit Station Operations
- Payment Recording
- Shift Management
- File-Based Data Persistence
- Reporting Functionality
- Console and Swing-Based Interfaces

## Technologies

- Java
- Object-Oriented Programming (OOP)
- Java Swing
- File Handling
- Layered Architecture
